# 在线书源仓库（临时测试不得外传）

本文档站在构建时会自动将官方书源发布到 `/booksources/` 路径下，可以作为**临时测试仓库**直接在 Legado Tauri 中添加使用。

## 仓库地址 (需要魔法上网才能访问)2

将以下地址复制到 Legado Tauri **书源管理 → 在线仓库** 的地址栏中，点击添加即可：

```
https://docs.legadoteam.org/booksources/repository.json
```

> **提示**：如果使用了自定义域名，把域名替换到上方地址的主机部分即可。

## 添加步骤

1. 打开 Legado Tauri，进入 **书源管理**
2. 切换到 **在线仓库** 标签页
3. 在地址栏粘贴上方 `repository.json` 地址，按回车确认
4. 仓库书源列表加载后，选择需要的书源点击安装
